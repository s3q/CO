What’s in the Download?

The non-source library files contain css & javavscrpt files only .. As for the sources, it contains disassembled sass files and css & javascript files, and there is also a complete css & javascript file.
Files & Folders What They Are Where You Should Start
dist / css All stylesheet for this library co.css or co.min.css
dist / js All javascript codes for this library co.js or co.min.js

Using Controlflex CSS

The css / co .css file contains the basic layout plus all the formatting you'll need when using controlflex. Next to it, you will find split files from the main file to be used if you want something specific and not all the formats. Copy link code and paste it between the two header tags ... and change the href value to the path to the CSS file you will find in the dist \ css \ co .css file. 
<link rel="stylesheet" href="your-path-to-controlflex/dist/css/co.min.css">

Using Controlflex JS

The js / co .js file contains javascript code along with all the formatting you'll need when using controlflex. Next to it, you will find fragmented files from the main file to be used if you want something specific from javascript code, and not all the codes. Copy and paste the script code before the end of the body tag ... and change the source value to the path to the javascript file which you will find in the dist \ css \ co .js file. 
<script src="your-path-to-controlflex/dist/js/co.min.js"></script>

Starter template

Be sure to have your pages set up with the latest design and development standards. That means using an HTML5 doctype and including a viewport meta tag for proper responsive behaviors. Put it all together and your pages should look like this:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <!--description-->
    <meta name="description" content="description......">

    <title>Document</title>

    <!--google font (optional) :-->  
    <link href="https://fonts.googleapis.com/css2?family=Nunito:ital,wght@0,200;0,300;0,400;0,600;0,700;0,800;0,900;1,200;1,300;1,400;1,600;1,700;1,800;1,900&display=swap" rel="stylesheet">

    <!--CONTROLFLEX (V.0.5)-->
    <!--CONTROLFLEX CSS-->
    <link rel="stylesheet" href="your-path-to-controlflex/dist/css/co.min.css">

    <!--window icon-->
    <link rel="icon" href="-----.png" size="32/32" type="image/png">

</head>
<body>

    <!--write-your-code-here-->

    <!--jQuery (3.5.1)-->
    <script src="https://code.jquery.com/jquery-3.5.1.min.js"></script>

    <!--CONTROLFLEX JAVASCRIPT-->  
    <script src="your-path-to-controlflex/dist/js/co.min.js"></script>
</body>
</html>
