المقدمة

ابدأ باستخدام مكتبة كونترول فلكس لإنشاء موقع حديث يتفوق على المواقع الأخرى من حيث الأداء والمظهر ، في وقت قياسي .. ابدأ في إعداد المستند للعمل مع المكتبة بشكل صحيح تنزيل حزمة ملفات المكتبة أولاً
  • ماذا يوجد في التنزيل؟
  • استخدام Controlflex CSS
  • استخدام Controlflex JS
  • نموذج المبتدئين

ماذا يوجد في التنزيل؟

تحتوي ملفات المكتبة بدون ملفات المصدر على ملفات css & javavscrpt فقط .. أما بالنسبة لملفات للمصادر ، فهي تحتوي على ملفات sass مفككة وملفات css و javascript ، وهناك أيضًا ملف كامل ملف css & javascript .
الملفات والمجلدات ما هم من أين يجب أن تبدأ
dist / css كل ورقة الأنماط لهذه المكتبة co.css or co.min.css
dist / js جميع رموز جافا سكريبت لهذه المكتبة co.js or co.min.js

استخدام Controlflex CSS

يحتوي ملف css / co .css على تخطيط الأساسي بالإضافة إلى كل التنسيق الذي ستحتاجه عند استخدام controlflex. بجانبه ، ستجد ملفات مقسمة من الملف الرئيسي لاستخدامها إذا كنت تريد شيئًا محددًا وليس كل التنسيقات. انسخ كود الرابط والصقه بين علامتي الرأس ... وغير قيمة href إلى المسار إلى ملف CSS الذي ستجده في dist \ css \ co .css . 
<link rel= "stylesheet" href="your-path-to-controlflex/dist/css/co.min.css">

إستخدم Controlflex JS

يحتوي ملف js / co .js على كود جافا سكريبت جنبًا إلى جنب مع كل التنسيقات التي ستحتاجها عند استخدام controlflex. بجانبه ، ستجد ملفات مجزأة من الملف الرئيسي لاستخدامها إذا كنت تريد شيئًا محددًا من كود جافا سكريبت ، وليس جميع الأكواد. انسخ رمز البرنامج النصي والصقه قبل نهاية علامة النص الأساسي ... وقم بتغيير القيمة المصدر إلى المسار إلى ملف javascript الذي ستجده في ملف dist \ css \ co .js . 
<script src="your-path-to-controlflex/dist/js/co.min.js"></script>

نموذج المبتدئين

تأكد من إعداد صفحاتك بأحدث معايير التصميم والتطوير. وهذا يعني استخدام HTML5 Dictype وتضمين علامة meta لإطار العرض للسلوكيات سريعة الاستجابة. ضع كل ذلك معًا وستظهر صفحاتك على النحو التالي: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <!--description-->
    <meta name="description" content="description......">

    <title>Document</title>

    <!--google font (optional) :-->  
    <link href="https://fonts.googleapis.com/css2?family=Nunito:ital,wght@0,200;0,300;0,400;0,600;0,700;0,800;0,900;1,200;1,300;1,400;1,600;1,700;1,800;1,900&display=swap" rel="stylesheet">

    <!--CONTROLFLEX (V.0.5)-->
    <!--CONTROLFLEX CSS-->
    <link rel="stylesheet" href="your-path-to-controlflex/dist/css/co.min.css">

    <!--window icon-->
    <link rel="icon" href="-----.png" size="32/32" type="image/png">

</head>
<body>

    <!--write-your-code-here-->

    <!--jQuery (3.5.1)-->
    <script src="https://code.jquery.com/jquery-3.5.1.min.js"></script>

    <!--CONTROLFLEX JAVASCRIPT-->  
    <script src="your-path-to-controlflex/dist/js/co.min.js"></script>
</body>
</html>
