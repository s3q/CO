About the library

First, this library is designed to facilitate the development of user interfaces, the part that sees the web page. This library helps you do this easily and in a short time without having to be professional in HTML & CSS languages, but I don't mean to know these two languages. If you want to start working with this library, you must first download the library from github and then go to the get start page to find out how to link your page with this library. You can check out a page about to learn morea page about to learn more about this library

Library-cans

You can find examples of this library in the examples page where it will display all the optimization of this library and all the new additions are put in an example. You can consider that examples are the best way to learn this library and its capabilities, also you will find an explanation on each example and also you will find how you can edit the files in order to come up with a new idea from this library and customize it for you as you want to deal with your site.
