Containers

Containers are a fundamental building block of Controlflex that contain, pad, and align your content within a given device or viewport.
  • The basic concepts
  • Container sizes
  • Default-container
  • Responsive container
  • liquid container
  • container-main

How they work

Containers are the most basic layout element in Controlflex and are required when using our default grid system. Containers are used to contain, pad, and (sometimes) center the content within them. While containers can be nested, most layouts do not require a nested container. Controlflex comes with three different containers:
  • .container, Which is width: 98% at all screen size until the last size (mxl: 1400px), where width is specified: 1400px for container.
  • .container-liquid, Which is width: 98% at all screen size until the last size (mxl: 1400px), where width is specified: 80% for container.
  • .container-main, Which is width: 100% at all screen size until the last size (mxl: 1400px), where width is specified: 1400px for container.
  • .container-res, which sets a width at each responsive screen size.

Container sizes

The table below illustrates how each container’s width compares to the original .container and .container-fluid across each screen size.
Class infix LXS XS SM MD LG XL MXL
.container 98% 98% 98% 98% 98% 98% 1400px
.container-liquid 98% 98% 98% 98% 98% 98% 80%
.container-main 100% 100% 100% 100% 100% 100% 1400px
.container-red 98% 98% 98% 740px 920px 1100px 1400px

Default-container

.container, Flexible container with all screens, and stop expanding in the last size
<div class="container">
<!-- Content here -->
</div>

Responsive container

.container-res, This class changes its width with every screen except for lxs, xs, sm. Where the width remains at 98%.
<div class="container-res">
<!-- Content here -->
</div>

Liquid container

.container-liquid, This container is used in the wide content .. in the screen size less than 1400px this container is similar to the original container .. After this size, its width becomes: 80%.
<div class="container-liquid">
<!-- Content here -->
</div>

Main container

.container-main, This container is very important in large screens, as this class can be placed in body tag.
<body class="container-main">
<!-- Content here -->
</body>
